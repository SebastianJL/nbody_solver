# N body project

## Data format
- data.txt\
  - No header
  - Data in columns\
    ID, Masses, x, y, z, Vx, Vy, Vz, softening, potential
- data.ascii
  - No header
  - Data in single column of length 8*N, N=number of particles.
  - x[i]
    y[i]
    z[i]
    Vx[i]
    Vy[i]
    Vz[i]
    softening[i]
    potential[i]
